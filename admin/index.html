Local AWS credentials

For signed or requester-pays scans in local Docker, keep refreshing temporary AWS credentials into the container. Replace your-profile, or omit AWS_PROFILE=... for your default profile. 
AWS_PROFILE=your-profile ./scripts/auto_refresh_credentials.sh
Show inline loop 
while true; do
  tmp_creds="$(mktemp /tmp/mapserver-creds.XXXXXX.json)"
  aws configure export-credentials --profile your-profile > "$tmp_creds" && \
    jq -e '.AccessKeyId and .SecretAccessKey' "$tmp_creds" >/dev/null && \
    docker cp "$tmp_creds" mapserver:/tmp/aws_credentials.json
  rm -f "$tmp_creds"
  sleep 900
done
